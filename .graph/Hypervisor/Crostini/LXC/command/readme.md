# Run LXC Command Directly
https://www.google.com/search?q=chromeos+termina+lxc+list

>AI Overview
>To list LXD containers in ChromeOS, open the Crosh shell (Ctrl+Alt+T), start the Termina VM using vmc start termina, and run lxc list. This command displays all containers, their status, IP addresses, and snapshots managed within the ChromeOS Linux environment. 

```
vmc start termina
lxc list
```
